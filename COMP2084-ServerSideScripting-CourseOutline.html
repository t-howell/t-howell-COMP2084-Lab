Course Description:

This course provides further development in designing and creating web sites that can create, read, update, and delete (CRUD) data from a database. The students will use web concepts, Object-Oriented Programming (OOP), relational database principles, and other server-side scripting to create web-based, data-driven applications.

Learning Outcomes:

Upon successful completion of this course, the student will have reliably demonstrated the ability to:

1. apply server-side scripting technologies and techniques to the development of web applications.
2. implement CRUD functionality through the application of server-side scripting
3. devise an authentication method, and manage a user session
4. create new objects from classes, using properties, methods and events
5. write advanced scripts which run directly on the server to generate dynamic content.

Course Content:

- Examining types of server-side scripting
- Review of relational database principles
- OOP components and techniques
- Web application security via authentication
- Producing dynamic web pages using current server-side
scripting technologies
